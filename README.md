# Weather Dashboard

A simple weather dashboard web application built with **JavaScript, HTML, and CSS** that retrieves and displays real-time weather data from an external REST API.

## Live Demo
https://muazshaikhh.github.io/weather-dashboard/

## Technologies
- JavaScript
- HTML
- CSS
- REST API (weather data)
- Asynchronous programming (async/await, Promises)

## Features
- Search for weather data by city
- Retrieve and display current weather conditions using a public REST API
- Handle API requests using asynchronous JavaScript
- Gracefully manage loading states and error handling
- Dynamically update the UI based on user input

## What I Learned
- Fetching and handling data from external APIs using JavaScript
- Managing asynchronous code with async/await and Promises
- Updating the user interface through DOM manipulation
- Structuring frontend code for readability and maintainability
